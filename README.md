# dbatools
DBA 助手工具

# 功能介绍：
  - 1.DBA健康巡检健康日报
  - 2 kafka topic增加 副本功能
  - 3 zabbix 自动添加主机。监控。模板
  - 4 zabbix 扩展告警收集，增加 电话，短信，邮箱，根据高级级别做调整
  - 5. otter 自动化配置
  - 6. 慢查询日报根据部门-项目-负责人。每日发送各个不同的团队
  - 7. 每日巡检数据库备份日报--获取备份是否成功
